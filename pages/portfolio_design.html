MODEL: UI CUBE [REDUNDANT]

Project Showcase Module

This 3D cube, built entirely with CSS, was the project showcase on the homepage in the first iteration of the portfolio. It rotated back and forth inside the water and paused when you hovered over it. If clicked, it would expand into a larger cube that displayed the projects in a slideshow. Inside, you could scroll through project details and use the navigation buttons to switch between them.

I added a subtle animated shine to the front face to give it a glass-like finish, which I have now re-used on this page which shows up occasionally.

MODEL: WORD CYLINDER [REDUNDANT]

Word Display Scroller

On the homepage, I wanted a neat way to list my technical skills. My idea was an 'infinite' scroller with words on a flat transparent surface. I created this originally with a cube and words on each side, which erased and rewrote themselves after each 'turn'.

This felt like overcomplicating things, so I tried to simplify it. My solution to this was to create a hexagon cylinder instead with the amount of faces scaling per word I wanted to display. This was the result of that idea.

MODEL: FLOATING ENTITY [REDUNDANT]

Interactive Contact Module

I designed this floating 3D rectangle as a unique way to present my contact information. It drifted gently across the water on the homepage using a JavaScript animation for its position and rotation. Clicking on it opened up a contact modal, just like this one. The animation also paused on hover to make it easier to click.

MODEL: 3D PRELOADER

Loading Animation

To make the initial page load more interesting, I created a loading screen featuring a spinning 3D version of my logo. To make it 3D, I've recreated it in an SVG and added depth to it by mirroring it on all the coordinates, as shown in this wireframe model.

I've added this rotating logo to the 'loading' screen as a way to show it off. This isn't an actual loading screen, as it only adds an arbitrary delay.

MODEL: BEAM OF LIGHT [REDUNDANT]

Lightbeam Effect

This is a custom lightbeam effect I created to add a bit of flair to the old homepage floater objects in the water. It created a 'beam of light' over the hovered element using CSS gradients and animated rays for a flickering effect.

I wanted to create a sort of 'ethereal' feel while also showing the user that those elements could be interacted with.

MODEL: PARALLAX BACKGROUND

Parallax Background

To simulate a 3D space on a 2D plane, I used a parallax effect that I learnt to do in a previous game project. The background is made of several image layers that move at different speeds when the camera moves along the scene. The layer positions are updated in the main animation loop based on the camera's movement and rotation.

MODEL: INTERACTIVE CONSTELLATION [REDUNDANT]

Interactive Night Sky

The old homepage's night sky was an HTML5 canvas animation that I wrote to show constellations on mouse hover. It also simulated 'supernovas' by making some stars explode at random intervals, which would disappear. Given enough time, it would eventually make the entire night sky pitch black.

This feature came with quite a few challenges. For example, resizing the window changed the canvas's dimensions and caused the hovering to be inaccurate. I also had to add a throttling feature as the amount of stars on the page would lagg the website. This throttling is still used in other parts of the code.
